# h5nx-reassortment-and-risk-assessment
This repo contains the code necessary to identify H5Nx reassortant clusters using replicate runs of [TreeSort](https://github.com/flu-crew/TreeSort). This approach first identifies non-reassorting groups of viruses from each TreeSort replicate, then utilizes hierarchical clustering to summarize groupings across replicates. In doing so, only highly supported reassortant clusters across many replicates are identified, preventing spurious groupings that can arise from single runs of TreeSort (due to variations in tree topology, time resolution, clock rate estimates, etc.).
